Blazing Fast Downloads

Powered by yt-dlp — the fastest and most feature-complete video downloader available, supporting thousands of sites.

📡

Live Stream Capture

Capture live streams in real time with ffmpeg integration. Never miss a broadcast again.

🎨

Beautiful UI

A sleek, dark-themed Electron interface built with React and Tailwind. Clean, distraction-free, and responsive.

🔌

Plugin System

Extend StreamDock with community plugins. Add new sources, processing steps, and output formats.

🛡️

Sandboxed & Safe

Electron security best practices applied throughout. IPC is typed and validated at every boundary.

🖥️

Cross-Platform

Native builds for Windows (NSIS), macOS (DMG), and Linux (AppImage) — one codebase, three platforms.

System Requirements

🪟
Windows

Windows 10 (64-bit) or later. No additional runtime required — everything is bundled.

🍎
macOS

macOS 12 Monterey or later (Apple Silicon & Intel supported).

🐧
Linux

Ubuntu 20.04 / Debian 11 or compatible. Mark AppImage as executable before running.

⚙️
Binaries

Place yt-dlp and ffmpeg in the binaries/ folder, or install on PATH.
